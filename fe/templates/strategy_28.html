What is HTTP Caching with ETags?

Unlike application-layer caches (Redis, in-memory), HTTP caching is built into the protocol itself. The server sets response headers (ETag, Last-Modified, Cache-Control) that instruct any HTTP-compliant client — browser, CDN, proxy — how to cache and revalidate. No Redis required.

ETag + If-None-Match
Server fingerprints each resource. Client sends the fingerprint back. Hash match → 304, zero body bytes.
Cache-Control
max-age, stale-while-revalidate, immutable — tell how long to trust the cached copy.
Bandwidth saving
A 304 carries headers only (< 200 bytes). Skipping the body of a 5 KB resource 1 000 times saves 4.8 MB per endpoint.

ETag Conditional Request Lifecycle

Three possible outcomes for every GET request

① First request
Client → GET /article/1
Server → 200 OK
   ETag: "a3f9b2c1…"
   Cache-Control: max-age=60
   Body: {full JSON}

Client stores the ETag.
② Conditional GET (unchanged)
Client → GET /article/1
   If-None-Match: "a3f9b2c1…"
Server checks: hash matches
Server → 304 Not Modified
   ETag: "a3f9b2c1…"
   Body: NONE (0 bytes)
③ Conditional GET (changed)
Content was updated — new ETag
Client → GET /article/1
   If-None-Match: "a3f9b2c1…"
Server: hash mismatch
Server → 200 OK
   ETag: "e7d1f4a2…" (new)
   Body: {updated JSON}

# be/routers/s28_http_cache.py
import hashlib, json
from datetime import datetime, timezone
from email.utils import format_datetime, parsedate_to_datetime
from fastapi import Request
from fastapi.responses import JSONResponse, Response

# Compute ETag from SHA-256 of sorted JSON serialisation
def _compute_etag(data: dict) -> str:
    s = json.dumps(data, sort_keys=True, default=str)
    return f'"{hashlib.sha256(s.encode()).hexdigest()[:16]}"'

@router.get("/article/{article_id}")
async def get_article(article_id: int, request: Request, db=Depends(get_db)):
    article = await db.get(NewsArticle, article_id)
    data     = _article_dict(article)
    etag     = _compute_etag(data)
    lm_str   = _fmt_last_modified(article.updated_at)

    # ── Conditional GET: ETag ──────────────────────────────
    if_none_match = request.headers.get("if-none-match")
    if if_none_match == etag:
        return Response(                              # 304 — no body!
            status_code=304,
            headers={"ETag": etag, "Cache-Control": "public, max-age=60..."},
        )

    # ── Full 200 response ──────────────────────────────────
    return JSONResponse(
        content=data,
        headers={
            "ETag": etag,
            "Last-Modified": lm_str,
            "Cache-Control": "public, max-age=60, stale-while-revalidate=30",
        },
    )
